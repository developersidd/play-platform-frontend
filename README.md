# 🎬 Play Video Platform

**Play Video Platform** is a modern video-sharing platform built with **Next.js**, **Tailwind CSS**, and **Shadcn**. It enables users to browse, search, watch, and upload videos — delivering a YouTube-like experience.

<img src="https://i.ibb.co/8D7rjT1j/home.png" alt="Home Page" width="100%" />

---

## 🚀 Features

- 🔐 Custom User authentication
- 🔄 PWA support
- 📤 Video upload & management
- 🔍 Search functionality
- 📱 Fully responsive design
- ▶️ Video playback with React Player
- 📝 Comments & likes
- 🎞️ Playlists & subscriptions
- 🌓 Dark mode support
- ⏳ Infinite scrolling for video lists
- 🕒 Video history tracking
- 📌 Watch later functionality
- 👤 User profiles with video statistics
- 📊 Dashboard with video analytics
- 🐦 Tweets integration

---

## 🛠️ Technologies Used

- **Next.js**
- **Tailwind CSS**
- **React Virtualized**
- **Shadcn UI**
- **Cloudinary** (for image & video storage)
- **React Player**
